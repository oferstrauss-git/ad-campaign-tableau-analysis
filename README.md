## Tableau Dashboards

### 1. Ad Campaign Performance Dashboard
Two-page analytical dashboard covering campaign efficiency,
Pareto analysis, scatter plot quadrant analysis and 
campaign activity patterns.

**Pages:**
- Performance Overview — KPI table, dynamic KPI time series, 
  CPC vs CVR scatter plot, Avg CPC by country
- Campaign Behaviour — Volume trends, Pareto analysis, 
  campaign efficiency, consecutive months activity
